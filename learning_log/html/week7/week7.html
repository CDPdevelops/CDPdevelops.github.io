Conditional Branch Statements – checking a text input field
The first step was creating an HTML page that collects a user's name using a text input field and calls a function to display the user's name with a welcome message only when the function is called using a button. Such as "*name*, Welcome to my homepage!".
Figure 2
A JavaScript if statement was incorporated, so that if the user inputs their name, the web site will display the welcome message. The webpage was saved as practical6_T1a.html.
Figure 2
The next step was to modify the code above to alert the user to enter their name if the 'fname' input field have been left blank. The alert message: "Please enter your name!". If the user entered their name the welcome message would proceed to screen. The webpage was saved as Practical6_T1b.html.
Figure 3
Conditional Branch Statements – checking radio button input
Task 2 involved the script Practical4_T1b.html to be modified so that if the user selects a university it will display the selected university information. If the visitor did not make the selection, an alert message will notify them to make a selection. The alert message: "Please select a university!". The webpage was saved as Practical6_T2a.html.
